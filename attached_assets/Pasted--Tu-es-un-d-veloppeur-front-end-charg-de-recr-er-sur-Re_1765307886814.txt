
Tu es un développeur front-end chargé de recréer sur Replit un site vitrine inspiré de https://congonaparis.fr pour l’événement « Congo Na Paris ».

========================
1. STACK & STRUCTURE
========================
- Utilise un projet simple HTML / CSS / JavaScript (pas besoin de framework).
- Arborescence :
  - index.html  (Accueil)
  - presentation.html
  - contact.html
  - programme.html
  - intervenants.html
  - partenaires.html
  - participer.html (page d’info générale / redirection stands + billetterie + partenariat)
  - magazine.html (liste des articles)
  - style.css (styles globaux)
  - main.js (petites interactions : menu mobile, scroll vers sections, bouton "back to top")

- Le design doit être **responsive** (desktop, tablette, mobile).

========================
2. IDENTITÉ GRAPHIQUE
========================
Couleurs (proches de l’original) :
- Bleu nuit principal : #050816 (fond de header, sections sombres)
- Rouge vif principal : #FF3333 (boutons, chiffres, tags)
- Blanc : #FFFFFF (texte et fonds clairs)
- Gris très clair : #F5F5F5 (fonds de blocs clairs)
- Violet foncé d’accent pour les pastilles / bullets : #3E2A54

Typographies :
- Police principale (titres et menus) : "Poppins", sans-serif
- Police secondaire (paragraphes) : "Inter" ou "Open Sans", sans-serif
- Espacements généreux, grands titres, tout en majuscules pour certains blocs (BILLETERIE, MAGAZINE, etc.).

Boutons :
- Forme : pilule / arrondis forts (border-radius: 999px).
- Couleur fond : rouge (#FF3333).
- Texte blanc en uppercase (ex : "BILLETERIE", "S’ABONNER").
- Effet hover : légère augmentation de luminosité + ombre douce.

Images :
- Utilise des images de placeholders libres (ou une couleur de fond) avec un filtre rouge / bleu nuit pour rappeler le site.
- Toutes les images doivent être en `object-fit: cover;` avec des coins légèrement arrondis.

========================
3. HEADER & NAVIGATION
========================
Présent sur TOUTES les pages.

Layout (desktop) :
- Barre blanche en haut, pleine largeur.
- À gauche : logo "CONGO NA PARIS" sur deux lignes, en texte stylisé (ou image si dispo).
- Au centre : menu horizontal.
- À droite : icône loupe (recherche) – pas besoin de vraie fonctionnalité, juste décorative.

Menu :
- Items : "Accueil", "A propos" (menu déroulant : "Présentation", "Contact"), "Edition 2025" (déroulant : "Programme", "Intervenants", "Partenaires"), "Participer", "CNP Mag", "CNP TV".
- CNP TV pointe vers une URL externe (par ex. YouTube).
- L’item de la page active a un petit soulignement rouge.

Version mobile :
- Logo à gauche, icône de menu (hamburger) à droite.
- Quand on clique sur le hamburger : menu en overlay ou en dropdown occupant la largeur.

========================

- Disposition : deux colonnes (texte à gauche, image à droite).

Texte à gauche :
- Petit tag en rouge (facultatif).

  "Premier salon
   socio-culturel
   congolais en
   Europe, de

- (Optionnel) deux petits points rouges pour la navigation de slides – décoratif uniquement.

4.2. Bandeau chiffres clés
- Section immédiatement sous le hero, fond rouge.
- 4 blocs centrés horizontalement :

  - "+130" — label "INTERVENANTS"
  - "6" — label "ÉDITIONS PRÉCÉDENTES"
- Gros chiffres blancs, label en uppercase plus petit.

4.3. Bloc "Pourquoi nous choisir ?"
Section sur fond blanc, deux colonnes :

Colonne gauche :
- Label rouge : "[CNP 7ᵉ ÉDITION]" (en petites caps).
- Titre noir : "Pourquoi nous choisir ?"
- Paragraphe de présentation (texte en français, peut être inspiré du site mais tu peux utiliser un texte de démonstration).

  - Rencontrer les acteurs clés de la diaspora et du Congo
  - Découvrir les opportunités économiques et culturelles

  - S’immerger dans une ambiance festive et constructive

Colonne droite :
- Grande image type conférence avec un bouton "play" rouge au centre (rond), comme une vidéo.

4.4. Bloc "Une 7ᵉ édition exceptionnelle / Billetterie"

4.5. Trois cartes (Programme / Devenir exposant / Regarder CNP TV)
Section juste sous la billetterie :

Carte 1 :

- Texte bref.

Carte 2 :
- Titre : "Devenir Exposant"
- Texte bref.

Carte 3 :
- Titre : "Regarder CNP TV"
- Texte bref.
- Bouton/lien vers la chaîne YouTube.

4.6. Section "MAGAZINE / CNP MAG"
- Fond gris très foncé (#151515 ~), grandes lettres "MAGAZINE" en arrière-plan (en énorme, en transparence).
- Titre centré blanc : "CNP MAG".
- Sous le titre : 3 cartes d’articles de blog.

Chaque carte :
- Image en haut.
- Tag rouge (par ex. "CULTURE & DIASPORA", "CNP MAG", "ÉVÉNEMENTS").
- Date type "[ 08/31/2025 ]".
- Titre en 2 lignes max.
- Paragraphe d’extrait (lorem ipsum).
- Les cartes renvoient vers magazine.html ou vers de faux articles.

4.7. Bouton "Back to top"
- En bas à droite, sur toutes les pages, un bouton rond rouge avec une flèche vers le haut.
- Position : fixed, bottom: 20px, right: 20px.
- Clique -> scroll smooth vers le haut.

========================
7. COMPORTEMENT & DÉTAILS TECHNIQUES
========================
- Scroll doux (CSS `scroll-behavior: smooth;` ou JS).
- Menu sticky : le header reste collé en haut quand on scroll.
- Animation légère au survol des cartes (scale 1.02 + ombre).

Accessibilité :
- Utiliser des balises sémantiques : <header>, <nav>, <main>, <section>, <footer>.
- Texte alternatif (`alt`) sur toutes les images.
- Contraste suffisant entre texte et fond.
